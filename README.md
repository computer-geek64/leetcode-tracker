## About

LeetCode Tracker provides a near-term dashboard that tracks users in a friendly problem-solving competition, ranking users by metrics that encourage consistency and dedication.
It updates leaderboard rankings by pulling recent submission history from the LeetCode API on a regular schedule, as well as on demand via rate-limited requests.
